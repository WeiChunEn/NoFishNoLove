# NoFishNoLove-Game
ビデオリンク：https://drive.google.com/open?id=1Nq-V-D6e6BU60a4mqN8CeAV4SWUQA7ad<br>
這個遊戲是關於兩隻公貓為了討好一隻愛吃魚的母貓互相競爭的遊戲<br>
兩位玩家各扮演一隻貓咪
誰先把頭上的漁量條集滿誰就勝利<br>
河中除了有不同分數的魚
還有有特殊效果的箱子及會干擾玩家的水母<br>
