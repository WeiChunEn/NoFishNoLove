# 為愛而釣-PC_Game
## ゲーム紹介
これは、2匹のオス猫が競い合って、魚が大好きなメス猫を喜ばせるのゲームです<br>
2人のプレーヤーがそれぞれ猫を演じます
最初に頭に魚の量を集めた人が勝つ<br>
川の中でスコアの異なる魚に加えて<br>
プレイヤーを邪魔するクラゲや、捕まったときに特別な効果を持つボックスもあります<br>
